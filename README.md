# local-ai-agent-anythingllm-docker

## INSTALL

https://peter-nhan.github.io/posts/Ollama-AnythingLLM/

## RUN

docker compose up -d

## Setup

### Download models

1. Open Ollama WebUI: http://localhost:3010/
2. Go to Settings -> Models
3. Download models, for example phi4

### Start AnythingLLM

http://localhost:3001/
